# Prática 1 - Calculadoras

Este projeto faz parte da disciplina INF311 e consiste em uma calculadora básica (Cálculadora A) e uma completa (Calculadora B) desenvolvida em Java + xml.

## Funcionalidades
* Soma e Subtração
* Interface adaptativa
* Validação de campos vazios

## Tecnologias
* Java
* Android Studio
* Componentes de Layout XML
